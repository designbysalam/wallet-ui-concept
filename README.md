# walletflow-digital-wallet

## Overview

walletflow-digital-wallet is a personal UI/UX design practice project created in Figma. The project focuses on designing a modern digital wallet interface that enables users to manage their finances, monitor transactions, and perform quick money transfers through a clean and intuitive mobile experience.

---

## Project Type

Personal Practice Project

---

## My Role

UI/UX Designer

---

## Tools Used

- Figma

---

## Features

- Wallet Dashboard
- Balance Overview
- Quick Money Transfer
- Transaction History
- Income & Expense Summary
- Trade Balance Overview

---

## Project Goal

The goal of this project was to practice designing a modern fintech dashboard with a strong emphasis on visual hierarchy, financial data presentation, and user-friendly navigation.
